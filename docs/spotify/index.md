# Spotify Setup

This section covers everything you need to configure Spotify integration with Spotify to Plex.

---

## Overview

To use Spotify to Plex, you need:

1. **Spotify API Credentials** - Required for accessing playlist data
2. **Authentication** - Connect your Spotify account for user-specific features
3. **User Integration** - Optional multi-user support
